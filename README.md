# Knight Project
**Auteur :** Mouhyadine Zakaria Djama

## Description
Knight Project est un jeu d'action de type "Endless Runner" développé en **C++** avec la bibliothèque **Raylib**. Vous incarnez un chevalier courageux qui doit traverser des terres hostiles, évitant les haches légendaires "Laeva" tout en récoltant des melons magiques pour survivre. Pour plus d'info sur l'histoire je vous invite a toucher la touche **S** une fois que vous rentrer dans l'executable pour voir l'affichage de l'histoire.

## Fonctionnalités Spéciales
Le jeu intègre des mécaniques de gameplay uniques liées à la progression :
* **Système de Melons :**
    * **3 Melons :** Déclenche un état d'**invincibilité** temporaire.
    * **6 Melons :** Octroie une **vie supplémentaire** au chevalier.
* **Aura de Perception (Slow Motion) :** Chaque fois que le chevalier se rapproche dangereusement d'un obstacle, le temps ralentit pour permettre des réflexes héroïques.
* **Système de Vies :** Affichage dynamique de la santé via des icônes de cœurs.

##  Commandes
* **ESPACE :** Sauter par-dessus les obstacles.
* **TOUCHE S :** S'accroupir pour passer sous les haches.
* **TOUCHE Q :** Couper la musique.
* **TOUCHE E :** Activer la musique.
* **ENTRÉE :** Démarrer le jeu ou recommencer.

## Comment lancer le jeu
1. Téléchargez l'intégralité du dépôt.
2. Assurez-vous que les fichiers `.dll` sont dans le même dossier que l'exécutable.
3. Lancez **`Knight.exe`**.
